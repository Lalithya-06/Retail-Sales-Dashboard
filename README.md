# 📊 Retail Sales & Profit Analysis Dashboard

## 📌 Project Overview
This project analyzes retail sales data to uncover insights on sales performance, profit trends, and customer behavior.

## 🛠 Tools Used
- Excel (Data Cleaning & Analysis)
- Power BI (Dashboard Creation)

## 📊 Dashboard Features
- Total Sales, Profit, Orders, Discount KPIs
- Sales by Region
- Sales by Product Category
- Top Products by Sales
- Monthly Sales Trend
- Profit by Customer Segment

## 🔧 Process
1. Data Cleaning in Excel
2. Data Analysis using Pivot Tables
3. Data Visualization in Power BI
